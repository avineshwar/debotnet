Debotnet - Source Code
Description  :  Initial release 23-12-2019

                Debotnet is a tiny portable tool for controlling Windows 10's many privacy-related settings
                and keep your personal data private.

                Debotnet requires Windows 10 including both 32-bit and 64-bit versions.

                More infos can be found on
                https://github.com/mirinsoft/debotnet

                Copyright (c) 2019 Mirinsoft


Deviations from the original sources
   - WGET.EXE is not included in this sources. Download and put it to bin folder of Debotnet to update/download third-party script files!!!
   - modFolderBrowser has been replaced by a simpler version from planet-source-code.com/Zhu JinYong
   - modStdout has been replaced by an improved version from http://www.planet-source-code.com/vb/scripts/ShowCode.asp?txtCodeId=35492&lngWId=1
   - Many new features planned for release 0.8/1.0 are not included in the sources, because I didn't finish it and removed them.
   - Sources provided here are based upon release 0.7.6 and refreshed with some comments
